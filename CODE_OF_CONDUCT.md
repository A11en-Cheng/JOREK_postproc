# 行为准则

## 我们的誓言

为了促进一个开放和热情的环境，我们作为贡献者和维护者承诺，无论年龄、身体特征、残疾、种族、性别认同和表现、经验水平、国籍、外表、种族、宗教或性身份和性取向如何，为项目和社区中的每个人提供一个无骚扰的体验。

## 我们的标准

有助于创建积极环境的行为包括：

- 使用欢迎和包容性语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 专注于对社区最有利的事情
- 对其他社区成员表示同情

不可接受的行为示例包括：

- 基于性的骚扰或不适当的性言论或图像
- 人身攻击、侮辱或贬低评论
- 公开或私下骚扰
- 发布他人的私人信息（例如地址或电子邮件），没有明确许可
- 在职业背景下的不适当行为
- 其他在专业环境中合理认为不适当的行为

## 执行责任

项目维护者负责澄清可接受行为的标准，并对任何不可接受行为采取适当和公正的纠正措施。

项目维护者有权并有责任移除、编辑或拒绝评论、提交、代码、wiki 编辑、问题和其他不符合本行为准则的贡献，或暂时或永久禁止其贡献者进行他们认为不适当、威胁、冒犯或有害的行为。

## 范围

本行为准则适用于所有项目空间，也适用于当个人代表项目或其社区时的公共空间。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体账户发布或在官方活动中充当任命代表。

## 执行

可以通过联系项目维护者 Allen Cheng (Allencheng@buaa.edu.cn) 报告不当行为。所有投诉将被审查和调查，并将导致被认为必要和适当的回应。

项目团队有义务对事件报告人的身份保密。特定执行政策的更多详情可能会另行发布。

不能真诚遵守或执行本行为准则的项目维护者可能面临由项目领导班子确定的暂时或永久处罚。

## 归因

本行为准则采用自 [Contributor Covenant][homepage]，版本 1.4，可在 https://www.contributor-covenant.org/version/1/4/code-of-conduct.html 获得。

有关常见问题，请参见 https://www.contributor-covenant.org/faq。
